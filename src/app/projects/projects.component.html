Projects

  • Thesis Project (In Progress): PhishAlert - AI-Enhanced URL Protection (Gmail Extension)

    PhishAlert is a browser extension for Gmail that detects phishing links in emails. It uses a Spring Boot backend to communicate between the frontend and analysis tools. URLs are analyzed using OpenAI for intelligent detection and the Google Safe Browsing API for threat checks. PhishAlert provides users with safety alerts, helping to prevent phishing attacks and protect sensitive information.
